# AIBetExplorer - Описание продукта

## Почему этот проект существует

AIBetExplorer создан для решения проблемы объективного анализа спортивных ставок на основе исторических данных. Традиционные методы прогнозирования часто опираются на интуицию или ограниченные данные, что приводит к субъективным решениям и потерям денег.

## Какие проблемы решает

1. **Субъективность прогнозов**: Заменяет интуитивные решения на статистически обоснованные расчеты
2. **Отсутствие исторических данных**: Автоматически собирает и накапливает статистику матчей из надежного источника (BetExplorer)
3. **Сложность анализа**: Предоставляет готовые инструменты для расчета вероятностей исходов и оптимальных размеров ставок
4. **Риск потери банка**: Использует критерий Келли для математически обоснованного управления капиталом

## Как это работает

### Сбор данных

1. Система автоматически загружает данные о матчах с BetExplorer.com
2. Парсит HTML-страницы и извлекает:
   - Результаты матчей (счет, таймы)
   - Информацию о командах (названия, эмблемы)
   - Коэффициенты букмекеров
   - Детали событий (голы, авторы, минуты)
3. Сохраняет все данные в PostgreSQL базу данных

### Анализ и прогнозирование

1. **Статистический анализ**: Рассчитывает предматчевую статистику для каждой команды
   - Среднее количество забитых/пропущенных голов
   - Процент побед, ничьих, поражений
   - Статистика дома/в гостях

2. **Рейтинг команд**: Вычисляет динамический рейтинг на основе:
   - Результатов матчей
   - Разницы голов
   - Домашнего преимущества

3. **Прогнозирование**: Комбинирует статистику и рейтинги для расчета вероятностей исходов
   - Победа домашней команды
   - Ничья
   - Победа гостевой команды

4. **Расчет ставок**: Применяет критерий Келли для определения оптимального размера ставки
   - Учитывает вероятность исхода
   - Учитывает коэффициент букмекера
   - Управляет рисками и банкроллом

## Опыт пользователя

### Для разработчика

1. **Настройка**: Конфигурация в `app/config_new.py`
   - Указание базы данных
   - Выбор видов спорта
   - Настройка каталогов

2. **Запуск загрузки данных**:
   ```bash
   python app/main.py
   ```

3. **Запуск анализа**:
   ```bash
   python app/analysis.py
   ```

### Для аналитика

1. Получает доступ к накопленной статистике через базу данных
2. Может настраивать параметры анализа (количество туров для учета)
3. Получает готовые прогнозы и рекомендации по ставкам
4. Может анализировать эффективность разных стратегий

## Целевая аудитория

- **Разработчики**: Могут использовать систему как основу для собственных решений
- **Аналитики ставок**: Получают инструмент для объективного анализа
- **Исследователи**: Имеют доступ к большим объемам структурированных данных

## Ключевые преимущества

1. **Автоматизация**: Полностью автоматический сбор и обновление данных
2. **Масштабируемость**: Поддержка множества видов спорта и чемпионатов
3. **Научный подход**: Использование проверенных математических методов
4. **Гибкость**: Возможность настройки параметров и стратегий
5. **Открытость**: Открытый исходный код, возможность модификации

## Ограничения

1. Зависимость от доступности BetExplorer.com
2. Требует наличия PostgreSQL базы данных
3. Не гарантирует прибыль (прогнозирование - это оценка вероятностей, не гарантия)
4. Требует технических навыков для настройки и использования
